# ToDo List Website

This is a simple ToDo list website created using HTML, CSS, and JavaScript. It allows users to add, remove, and mark tasks as completed.

## Features

- Add tasks: Users can add tasks by typing in the input field and pressing Enter or clicking the "Add" button.
- Remove tasks: Users can remove tasks by clicking the delete button associated with each task.
- Mark tasks as completed: Users can mark tasks as completed by clicking the checkbox next to each task.

## How to Use

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/KAMRANZAFAR4343/todo-list.git
   ```

2. Navigate to the project directory:

   ```bash
   cd todo-list
   ```

3. Open the `index.html` file in your web browser to view the ToDo list website.
